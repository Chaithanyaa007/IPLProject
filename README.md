##  Overview
This project predicts the winner of Indian Premier League (IPL) matches using Machine Learning.

The model is trained on historical IPL match data and predicts match outcomes based on team and toss information.

---

##  Problem Statement
To build a classification model that predicts the winning team of an IPL match using historical match data.

---

##  Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
- Jupyter Notebook

---

##  Features Used
- Team 1
- Team 2
- Toss Winner
- Toss Decision

---

##  Model Used
Random Forest Classifier
